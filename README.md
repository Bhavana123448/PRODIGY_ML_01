Task 01 - House Price Prediction Using Linear Regression

### Objective

The objective of this task was to implement a Linear Regression model to predict house prices based on square footage, number of bedrooms, and number of bathrooms.

### Tools and Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

### Dataset Used

House Sales Dataset containing 9,007 housing records and multiple property-related features.

### Approach

* Loaded and explored the housing dataset.
* Performed data preprocessing and checked for missing values.
* Selected square footage, bedrooms, and bathrooms as predictor variables.
* Conducted exploratory data analysis using various visualizations.
* Split the dataset into training and testing sets.
* Built a Multiple Linear Regression model using Scikit-learn.
* Predicted house prices based on housing features.
* Evaluated model performance using MAE, RMSE, and R² Score.
* Visualized feature relationships and model predictions through graphs and charts.

### Key Findings

* Square footage showed the strongest relationship with house prices.
* Bedrooms and bathrooms also contributed to price prediction.
* The model achieved an R² Score of approximately 0.48.
* The trained model successfully predicted house prices for new housing data.

### Conclusion

This project demonstrated how Machine Learning and Linear Regression can be applied to solve real-world house price prediction problems. It provided practical experience in data preprocessing, exploratory data analysis, model training, evaluation, and predictive analytics.
